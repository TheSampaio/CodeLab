## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/weather-forecast-spa.git
   cd weather-forecast-spa
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```
